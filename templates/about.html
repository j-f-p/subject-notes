About

This project is a web application for collaboratively compiling notes about various topics of a subject. The demonstration notes are taken from a book about Deep Learning. This app can authenticate any user with a Google account. Authenticated users are able to post, edit and delete notes. Components of this project were inspired by Udacity courses about web-server app development with Flask. This was the second of three primary projects completed toward achieving the Full Stack Web Developer Nanodegree.
